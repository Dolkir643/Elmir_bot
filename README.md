# Портфель ЭМ — Telegram-бот (HAOS)

Данные берутся с листа **«Бренды»** в Excel: у каждого бренда своя строка и почта; направления и группы задаются строками-заголовками выше.

**Ответ бота:** только **адрес(а) почты**.

Поиск: **бренд** (с учётом алиасов и транслита) → **группа** → **направление**. Поддерживаются русское/английское написание и типичные опечатки (например: Shure, Шур, шур, Shur, Шуре → одна и та же почта). Варианты задаются в `em_portfolio_bot/brand_aliases.json` и автоматической транслитерацией.

## Excel

В книге должен быть лист **`Бренды`** (как у вас): колонка A — название, B — почта. Направления и группы — отдельные строки (списки имён в `scripts/build_vendors_from_xlsx.py`: `DIRECTIONS`, `GROUPS`). При новых группах в файле добавьте название в `GROUPS`.

## Сборка `vendors.json`

```bash
python3 scripts/build_vendors_from_xlsx.py "/путь/к/Портфель ЭМ для партнеров.xlsx" em_portfolio_bot/vendors.json
python3 scripts/export_positions_md.py   # опционально: таблица в docs/
```

Установка в HA — как раньше: репозиторий, `bot_token`, при необходимости `telegram_proxy`.

Основа подключения — как у [carvector-bot](https://github.com/Dolkir643/carvector-bot).
